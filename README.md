# KeyBoardDemo
## 1. 简介
本篇博客将实现一个自定义软键盘实现数字录入的小Demo，也是为之后可能遇到的输入手机号，支付键盘等做一些知识上的储备，也是希望能够帮到处于学习的同学，
在写的不对的地方希望各位能够指出来。含有demo。
